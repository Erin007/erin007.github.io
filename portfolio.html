Week 1: Calculator

We wrote calculator programs that could add, multiply, subtract and divide two numbers. Okay, so it's no T1-89 (man, I'm old), but it sure felt like a jump from the "Hello World" program I had only recently mastered. This project was the first where I...

• dealt with error handling for user input (albeit in a rudimentary way)

• refactored a long, long list of if statements into a more readable case statement

• used an unless loop to let a user continue "playing"

After meeting the baseline requirements, I wrote a little code that found factors of numbers, including negatives and identified prime numbers.
