Week 1: Calculator

We wrote calculator programs that could add, multiply, subtract and divide two numbers. Okay, so it's no T1-89, but it sure felt like a jump from the "Hello World" program I had only recently mastered. This project was the first where I...

• dealt with error handling for user input (albeit in a rudimentary way)

• refactored a long, long list of if statements into a more readable case statement

• used an unless loop to let a user continue "playing"

After meeting the baseline requirements, I wrote a little code that found factors of numbers, including negatives and identified prime numbers.

Week 2: Solar System

This week was a meteor shower of new information. We worked on a planetary project wherein we created objects in a planet class that have a variety of attributes. We completed the project in two waves...

• Wave 1: was the first time I used classes, and initialized new objects

• Wave 2: stepped it up a notch by asking us to use an array of planets to initialize a solar system object of a new class

• I also tackled some optional enhancements such as calculating the distance between two planets using their distance from the sun attributes. It was out of this world!

Through completing this project, I finally understand why everyone keeps harping on the idea that "everything is an object" in Ruby.

Week 3: Bank Accounts

Using a Bank module, we created various classes for different aspects of bank accounts. The assignment started out straightforaward enough, until objects of different classes had to associate with one another based on shared attributes. Whoa. Takeaways:

• Class inheritance saves you from repetition. It's super!

• I figured out how to raise Exceptions with useful messages. Don't even think about opening that Money Market account with less than $10,000.

• sprintf "%.2f" is a beautiful little snippet of code

You can take a peak at the final code here, and see my first ever class methods and read of info from a csv. Adios, gets.chomp. You can also see the comment relics of my first great "end" scavenger hunt. I'm totally ready to make secure accounts for a real bank, right?
