Week 1: Calculator

Okay, so it's no T1-89 (man, I'm old), but it sure felt like a jump from the "Hello World" program I had only recently mastered. This project was notable for:

  • being the first time I dealt with error handling for user input (albeit in a rudimentary way)
  • refactored a long, long list of if statements into a cleaner case statement
  • used an unless loop to let a user continue "playing"

Also, my inner math teacher/nerd came out and I went outside the requirements of the assignment to write a little code that could find factor negative numbers.
